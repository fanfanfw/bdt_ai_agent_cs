Integration Guide
Option 1: Embed Chat Widget

Add this iframe to your website:

<!-- AI Chat Widget -->
<iframe 
  src="{{ request.scheme }}://{{ request.get_host }}/api/widget/?api_key={{ assistant.api_key }}" 
  style="position: fixed; bottom: 20px; right: 20px; width: 400px; height: 600px; border: none; z-index: 1000;"
  title="AI Chat Assistant">
</iframe>
Option 2: API Integration

Use our REST API for custom integration:

POST {{ request.scheme }}://{{ request.get_host }}/api/chat/
Content-Type: application/json
Authorization: Bearer {{ assistant.api_key }}

{
  "message": "Hello, I need help",
  "session_id": "optional-session-id"
}

voiceChat.startListening();
